# ft_malloc

> Allocateur mémoire personnalisé — Projet 42

## Description

Réimplémentation de `malloc`, `free` et `realloc` sous forme de bibliothèque partagée (`libft_malloc.so`).

Gestion des zones mémoire (TINY, SMALL, LARGE), alignement et réutilisation des blocs libérés.

## Compilation

```bash
make
```

Produit `libft_malloc_<arch>_<os>.so` avec lien symbolique `libft_malloc.so`.

## Compétences

`C` · `Gestion mémoire` · `Bibliothèques dynamiques` · `Système`
